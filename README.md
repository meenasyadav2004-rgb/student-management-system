Student Management System 

Project Overview
This **Student Management System** is a console-based Python application that allows users to manage student records efficiently using **CRUD operations**. It connects to a **MySQL database** to store and retrieve student information and provides a simple, menu-driven interface for interaction.  

This project demonstrates the integration of Python with SQL, handling databases, and building a functional, real-world application.  

 Features
- **Add Student:** Input new student information (name, age, course, email) and save it to the database.  
- **View Students:** Display all student records stored in the MySQL database.  
- **Update Student:** Modify details (e.g., course) for existing students using their unique ID.  
- **Delete Student:** Remove student records from the database by ID.  
- **Menu-driven Interface:** Easy-to-use console menu for seamless navigation.  

 Tech Stack
- **Python 3.14** – Application logic and interface  
- **MySQL** – Database management and storage  
- **Git** – Version control for tracking changes  
- **MySQL Workbench / CLI** – Database creation and management
